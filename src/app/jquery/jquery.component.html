HTML P, SPAN, B, I, U Tags

What is Paragraph Tag?
The `< p>` tag in HTML is used to define a paragraph. It is a block-level element that creates a new paragraph in the document. The content inside the `< p>` tag is usually text, but it can also include other HTML elements such as links, images, and lists. Here is an example of how to use the `< p>` tag in HTML: < p>This is a paragraph of text.< /p>

What is Bold Tag?
The `< b>` tag in HTML is a formatting element used to make text bold. It is a inline-level element, which means it doesn't create a new line in the document. When the web page is rendered in a browser, the text enclosed within the `< b>` tags will be displayed in a bold font. Here is an example of how to use the `< b>` tag in HTML: < p>This is < b>bold< /b> text.< /p>

What is U Tags ?

The `< u>` tag in HTML is used to underline text. It is a formatting tag that is used to indicate to the browser that the enclosed text should be underlined. Here's an example of how to use the `< u>` tag:

< p>This is some < u>underlined< /u> text.< /p>

What is SPAN Tags ?

The `< span>` tag in HTML is an inline element that is used to group and apply styles to inline elements or text within a larger block of content. It is a very versatile tag that can be used for a variety of purposes, such as grouping text for styling, applying CSS styles to a specific section of text, or targeting a specific element with JavaScript. Here's an example of how to use the `< span>` tag:

< p>This is a < span style="color: red;">red< /span> word in a sentence.< /p>

What is i Tags ?

The `< i>` tag in HTML is used to indicate text that should be displayed in italicized or emphasized form. It is a formatting tag that is used to indicate to the browser that the enclosed text should be displayed in italics. Here's an example of how to use the `< i>` tag:

< p>This is some < i>italicized< /i> text.< /p>
