HTML Headings

What is Headings in HTML?

Headings in HTML (H1-H6) are used to define the hierarchical structure and organization of content on a web page. The H1 element is used for the main heading of the page, while H2, H3, H4, H5, and H6 are used for subheadings.

< !DOCTYPE html >
< html >
< head >
< title > TechEng < /title >
< /head >
<body >
< h1> This is Headings 1. < /h1>
< h2> This is Headings 2. < /h2>
< h3> This is Headings 3. < /h3>
< h4> This is Headings 4. < /h4>
< h5> This is Headings 5. < /h5>
< h6> This is Headings 2. < /h6>
</body >
</html >
