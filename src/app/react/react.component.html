HTML Elements

What is HTML Elements ?
HTML elements are the building blocks of a web page. They are the tags that define the structure and content of a web page. Here are some common HTML elements:
1.`< html>` - The root element that contains all other elements on the page.
2. `< head>` - Contains meta information about the page, such as the title, styles, and scripts.
3. `< body>` - Contains the visible content of the page, such as text, images, and videos.
4. `< h1> to < h6>` - Headings that define the hierarchy of the page content.
5. `< p>` - Paragraphs of text.
6. `< a>` - Links to other web pages or resources.
7. `< img>` - Images to be displayed on the page.
8. `< ul> and < ol>` - Unordered and ordered lists.
9. `< li>` - List items within a list.
10. `< table>` - A table with rows and columns of data.
11. `< tr>` - A table row.
12. `< th> and < td>` - Table header and data cells, respectively.
13. `< form>` - A form for user input, such as login or contact forms.
14. `< input>` - Input fields within a form, such as text boxes and radio buttons.
15. `< button> `- A clickable button on the page.
These are just a few examples of HTML elements. There are many more elements that can be used to create a web page with various content and functionality.
